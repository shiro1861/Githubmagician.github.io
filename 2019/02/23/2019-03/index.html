2019-03

2月 23, 2019

2019-02-23

出差归来。

和团队及客户相处耗费了我大量的意志力。

去的时候，听《晓说》采访韩寒第一集，名字就叫“我所理解的生活”，我记得是他以前写的一篇文章的名字。我也算是看着韩寒长大的一代人，并非狂热地喜欢他的文字，也没有被深刻地影响，但通过韩寒，我开始读钱钟书。所以《围城》对我影响更甚，有空没空，每年都要读一次。韩寒的《三重门》很好，给高中压抑的生活增添了很多幽默感。我从这两位作家身上学到的就是，在再不如意的时刻，都要勇于自嘲，并对生活放肆地哈哈大笑。

去候机，在航站楼一间还算大的书店买了《大人的日本史》，二十万字两小时解决。也就是网文体的日本近代简史，细节不算有趣，充其量算轻快版历史课本。没太多亮点。可能全书最深刻的就是腰封上引用的西塞罗的名言。买的原因也就是我随手一翻看见了木户孝允。

现场工作一例效率低下，得迁就着别人的工作习惯，被别人的情绪感染，还得对不喜欢的同事保持友善。好在一人住一间房，晚上回去翘着腿看《迷人的材料》，觉得还挺有意思，是没涉及过的领域。

回来的路上看在另一个航站楼购物区大书店买的《古文观止》，周文一例看得瞌睡，汉文读《史记》尚算有趣，一些谏言又看得瞌睡。再读苏轼的文章，又有新的感慨，《前赤壁赋》中“盖将自其变者而观之，则天地曾不能以一瞬。自其不变者而观之，则物与我皆无尽也，而又何羡乎！”，怎么读怎么对，还有点唯物的感觉。又想将《苏东坡传》或苏东坡的文字找出来读个一两遍。这些古老的智慧怎么就这么智慧啊。看累了古文，看手机kindle里房龙的《人类的故事》，前几章讲史前的还以为又是历史课本流水账，但越读越有趣，打算购入正本了。

下飞机笨拙地拿行李，邻座的勤奋工作大哥顺手帮拿下来，交换一个感谢的微笑。飞快走出油烟味弥漫的机场。

地铁上听高晓松采访韩寒第二集，也就是两个已近中年的人，回忆年少成名的事。人越老越觉得韩寒活得洒脱。他的文字不见得多么有文学上的价值，但他已经用自己喜欢的方式度过了自己的人生。我非常羡慕。

回家后吃饱喝足，买了新的数位板，也就是一个莫名其妙的灵感。

虽然说是要省钱凑读书的机票，但洗澡打开水龙头的一刻前，又觉得车到山前必有路，何必过多焦虑。紧接着思考了换城市生活的计划，但又觉得尚未到那个时点，切换城市不是小事，不需要太多的浪漫主义情怀，何况这几个大城市，只能一个比一个不近人情与冷漠。我为自己设置的条件就是，除非薪资与职业发展上有极大飞跃，否则无需在城市上过多纠结。与其说是纠结城市，不如说我只是纠结我需要和亲人一起生活，并负担较高的房租。说白了，我也只是自私地想过自己的小日子。（毕竟理想的生活空间，也就是三十平的小屋子，有个小榻榻米，一个小音箱，一个小冰箱，一个小阳台，宽敞的书桌，乱七八糟摞着的书。我也不喜厨房烟火，所以能舒服地窝着神游就好）。

出差之前见过一位好友。已俨然准备进入另外的行业，从他嘴里听出很多对互联网行业的憧憬，对大数据的吹捧，什么产品理念，营销理念诸如此类。开眼界确实开了，但我也不甚同意就是了（但也绝不妨碍我们是好朋友）。这些话没和他说过，我只是觉得，现在这些风口浪尖的东西，看起来好像光鲜亮丽，但究竟根基在哪，创造了什么新的价值还是开创了更有益的生活方式？这样说或许过于苛责，但这些概念、包装或者营销，确实有创造出什么吗？回顾风起云涌的千禧年，又觉得当时的青年们，确实创造了一些很实在的，能够改变世界的力量；但要我说现在这些互联网中产生的东西，并不见得将我们带向了一个好的方向。

沉迷于其中的我们，好像变得更加迟钝更加愚蠢。想起韩寒在访谈里说的，大意是，舆论不需要真相，舆论只需要情绪，人们不在意对错。

这些大数据和监控只能令我毛骨悚然罢了，什么精准投放，量身定制，不过是在将人当做家畜驯养。投其所好，徐徐图之，而后杀之。反正人嘛，一波来了一波又走，物我皆无穷嘛。

又想起韩寒那段podcast里面说的，有自知之明的人，就不应活在别人设定的判断标准之中。

这也算是我对现在社会的一点粗鄙兼幼稚的看法罢了。我始终有一个理念，世界在变，总有不变，紧抓那一点便不会被裹挟着走。或者尽管被裹挟而走，但仍能根据某些特定的锚点判断目前的行进情况。所以具体到行业选择，越接近这些锚点的工作，越能触碰这些核心的职位，就越不会被淘汰。这些锚点就是残存的人情味啊。越活着越觉得Blade Runner和攻壳机动队里描述的世界离我其实也并没很远了，人越活越虚幻。那些生活的实感——放学后成群结队吵吵嚷嚷的日子，凑钱买漫画杂志小说的日子，写完作业看电视的日子还有谁谁谁生日大家围坐在草地上聊天的日子——实在少得可怜。

友情爱情和亲情，稀薄得像喜马拉雅的空气。

我相信我的判断是对的，作为人，终究还是要去找这些锚点。是这些锚点，将支离破碎的人生片段连接在一起。

2019-02-24

看了《深夜食堂2》，是之前一次在健身房有氧训练的时候看的。一直不会主动看这种平缓的电影，总觉得会看不下去。还是有些比较戳泪点的地方吧，比如那个敲响风铃的瞬间，“叮”的一声，然后风铃摊子的老板转头和那对情侣说“人就要活得像这样”；再有就是从九州来到东京的老太太，在出租车里远远地看着自己抛弃的儿子，看着他和他的妻子孩子。

怎么说，觉得平淡的剧情里，还是有设计得比较巧妙用心的地方。但讽刺的是，它就像一个都市童话，而童话里的，往往是我们想要而得不到的东西。
