magician's log W1

8月 13, 2017

打算把魔法研习日记转换成周记，不想噜苏那么多。

进度

病好后的这周放弃了看《三国演义》，转为看《未来简史》，四天后的进度是82%，虽然一开始看的时候心潮澎湃有发现新大陆的感觉，不过接近尾声处又觉整本书基调悲观令人绝望，而且我觉得书里的句子给人一种傲慢的感觉，比如这样的句子：

21世纪经济学最重要的问题，可能就是多余的人能有什么功用。

锵锵三人行的podcast也评价了这本书，里面有个观点还挺妙的，说是因为现代科技的算法和数据最终会集聚在少数精英手里，而一旦这些少数人依附于某权利集团，那么《1984》的情形出现也并非不可能。

看《未来简史》的体验还是不错的，作者的写作风格就是非常干净的“论点+论据”型，非常GMAT，非常适合整理思维导图。尽管这本书读起来很启发，里面的观点也很抓人眼球，但就是充满了一种不能掩饰的优越感。所以我还是更欣赏该作者的另一本书《人类简史》，以及KK的《失控》。本来想买纸质版来整理一下的，不过现在想想我还是宁愿买《失控》。

因为看得太抑郁，打算看点亲和的书，于是转战《给青年诗人的信》。不知道看的什么译版，觉得用词和语句冗杂且难以理解，可能因为我不是青年诗人，不知道原文究竟是怎样的，不过好在篇幅比较短不至于难以忍受。整本书充满了那种虽然令人感到莫名其妙但读起来又觉得似曾相识说出了心中所想的句子，比如：

这是可笑的。我在这地坐在我的小屋里，我，布里格，已经是二十八岁了，没有人知道我这个人。我坐在这里，我是虚无。然而这个虚无开始想了，在五层楼上，一个灰色的巴黎的下午，它得出这样的思想：这是可能的吗，它想，人们还不曾看见过、认识过、说出过真实的与重要的事物？这是可能的吗，人们已经有了几千年的时间去观看、沉思、记载，而他们让这几千年过去了像是学校里休息的时间，在这时间内吃了一块黄油面包和一个苹果？ 是的，这是可能的。

好的，用短暂的复习前20分钟写了一些读后感，复习去也。

用了一个失眠的晚上看完了《给青年诗人的信》之后，因为想看些催眠的书，转战梁启超的《李鸿章传》，两日后进度50%，目前的观感是梁启超写得很够味，好肤浅的观感……好吧，其实观感是，“世事波上舟，沿洄安得住。”也是看得好抑郁，下本书真想看点欢快的。

魔法研习进度

这周五做了短暂休整，再次研究了有关记忆方法的文章，打算对学习方法进行调整。材料综合看了知乎上面一些大牛的经验，然后下载了一份有关记忆宫殿的学习材料，花了大约一个下午整理和阅读，并选取了部分来试验。整个过程还有点热血沸腾，莫名其妙……

目前最喜欢的还是串联记忆和联想记忆法，拿背诵内容多的审计实验了一下，比单纯背有意思多了，精神也更集中，因为要想莫名其妙的联想来记住。定桩和数字联想法有点不明所以，可能是对于我目前的备考科目不合适。

目前实践的心得：

  1. 记忆到考试就是“输入-编码-提取-运用”的过程，记不住实际上是编码和提取环节出现脱节
  2. 关于编码，就像是“制造路径”的感觉，我对应到使用思维导图整理这一步
  3. 提取就是以各种不同于初始输入的方式来复习
  4. 运用对于我就是做题

首先我改进记忆方法的目标（动机）是：

  • 第一目标：帮助我更高效地复习并通过以后所有我想通过的考试，一是专业知识，二是语言考试
  • 第二目标：在阅读和看电影时记忆更多的细节，一可写作时引用二可交谈时不至于空无一物
  • 第三目标：在将要开始的工作中能更快速地记下客户和同事的名字等细节以及对方交代的事务
  • 最高目标（误）：成为高段位学习者（坦白讲这个目标比拿下注册XXX考试的更令我热血沸腾是怎么回事……）

根据第一目标识别出的记忆客体特点：

  • 会计

    • 单纯记忆，概念性知识，分类标准等 （比如哪些交易计入什么科目）
    • 会计分录名称，需要一定理解
    • 一整套的处理思路，需要深入理解准则（某种交易从头到尾的处理）
    • 计算时易错的细节
    • 选择题快速计算的公式和技巧

  • 财务管理

    • 公式！！！
    • 解题思路
    • 单纯记忆，概念性知识，分类标准等
    • 计算时易错的细节

  • 税法

    • 征税对象税率征税时间计算公式
    • 计算细节

  • 审计

    • 单纯记忆，概念性知识等（应付简答题）
    • 需要深入理解并运用于判断的规则（比如选择什么审计程序，采用该程序或处理方法是否恰当）
    • 流程性质

  • 经济法就是往死里背吧（仲未复习）

  • 战略管理就是往死里背吧（仲未复习）

先写到这里，去干正事儿。突然感到谜之平静是怎么回事……

然后这几天复习进度都还算勉强踩着，审计一刷和财管二刷的进度差不多1.5天一章，包括复习讲义和自我归纳，会计卡死在负债和所有者权益那章，太庞杂了，正在一点点重新整理导图，税法卡死在企业所得税，也是庞杂……

反正现在的心情是很平静，感觉就像是准备一个庞大的期末考。上午基本可以保证复习效率，不过下午就有点悬。然后个人觉得自己的缺点就是太想一步到位，对自己每个复习环节要求很高，往往在开始前会觉得自己达不到而放弃去开始，造成拖延。下周的目标就是如何控制好下午的复习时间以及不要过于急进，学会分解任务，每次先做到及格。

病休筆記

8月 4, 2017

上次在日記里寫希望第二天好起來，結果並沒有好起來啊！被高高低低的燒連續折磨，因為頭痛只能看無聊的電視劇和電影度日，嚴重缺乏睡眠。

真的我覺得電視節目的設計機理是假設觀眾是白癡還是怎麼的。不過我居然看完了大軍師司馬懿之軍師聯盟，吳秀波還挺帥啊。

懷舊地看了中央六套的電影，我居然很懷念這裡的中文配音，有病啊。

西片的話看了這些：家長指導（Parental Guidance），又在家庭劇里發現新樂趣。還看了現代愛情故事（Frankie and Johnny），因為很喜歡開頭那種九十年代膠片色調，以為是部不錯的篇，結果只有阿爾帕西諾演得變態還能看啊。還有為奴十二年（12 Years a Slave），一開始以為法鯊演的黑奴我還在想誰是，結果又是演的變態嗎。裡面黑人老嬤嬤的演唱好聽。有些靜態的鏡頭感情張力和傳情達意的功夫很強，比如所羅門被吊在樹上那一幕。最後是李安的綠巨人（Hulk），說真的沒有那麼爛，我覺得李安是有自己的理解在裡面的，覺得基調是很悲愴的。各種打打殺殺場景有節制，其實我不太喜歡超級英雄片裡面炫技的場景。

國產片看了兩部後會無期（The Continent），歌還不錯吧，整個片子就像韓寒每本書的大合集，不過感覺雛形應該是1988：我想和這個世界談談？話說韓寒還真是喜歡那個煮青蛙的情節。片中江河用一鍋溫水煮青蛙，然後不停升溫，試圖告訴浩漢一旦溫度足夠高，動物會自己適應并挑出來，緊接著鏡頭就是浩漢一個鍋蓋蓋下來，“這才是現實”。其實我從高一開始有看韓寒的書，在那些苦悶的日子里確實對我而言是十分有趣的消遣，現在不會特地找他的作品來看，但也並不討厭，也許還會看。覺得對下面這段話挺有感觸的。

你們從小就認識，她卻早早離開故鄉。你去到她的城市，以為她過上理想日子，她的生活卻不過如是。你以為你能投靠，但是她自身難保。你以為你們能暢談往昔，卻越說越詞不達意。你們依然記得和掛念，只是不能抗衡時間和空間。你們最終告別，人生只是下一個地方和下一場戲。

還看了一部改編自亦舒小說喜寶的同名電影，真的，被裡面角色的世界觀嚇到了，覺得一屋子的歇斯底里症吧……

觀影記錄大體如此吧。真的，病了才知道珍惜健康啊。因為高熱不退，又沒什麼具體原因，老疑神疑鬼得了什麼不治之症，這才覺得人生苦短，該多干點正事，對自己負責啊。

magician今天有點長

7月 28, 2017

今早覺得頭痛，怎么都不想起床。堅持學了一個上午所有者權益和負債，實在不行了，於是休學罷工。發現現在真是沒有高中那股“讓我學我還能學啊啊啊啊還不能閉眼”的勁頭了。

下午看完了步履不停（歩いても 歩いても），看的是枝裕和導演的第一部片。原聲帶的吉他好聽，是夏天躺在陰涼的臥室里吹風扇的感覺。最後男主角的旁白令人有些難過，發現一個人的生老病死，說出來也就輕描淡寫一兩句話。

不過全片節奏對於我來說還是太緩慢。我也看過李安前期的“父親三部曲”（《推手》《飲食男女》《喜宴》），感覺要活潑很多:D，戲劇衝突感更強，令我在家庭劇中找到了意外的興趣點。

之後本來想重看銀翼殺手（Blade Runner），覺得太費神就刷完了涼宮春日的憂鬱（涼宮ハルヒの憂鬱）。这是我初中的時候差點就看了的作品。雖然自詡從中二少年圈退休多年，明知人生不复少年时，还是会看得热血沸腾，也还是希望自己能够回去。

其實也很久沒看動畫了，只是最近温习魔法太苦悶，挖了些初高中時期仰慕已久的舊作出來看。我看涼宮覺得比較有共鳴是在第十集左右。女主角涼宮有段自白大意是，曾經以為自己多麼獨特，但萬千世界人生海海，發現自己也只是普通人而已。發現這點之後，覺得人生陷入了憂鬱，所以才會希望每天的生活發生一些有趣的事。

這段自白，真是中二病開山鼻祖般的發言，我覺得說出了我理解的日本動畫的核心觀點之一──就算我獨特的存在與世界相矛盾，我也絕不妥協。雖然我個人對涼宮霸道的人設有一點不接受，但这个角色的魅力，就像片中男主角阿虛所說的“你們只會對現狀抱怨不滿，但春日是真的行動起來去改變這一切了啊。”
所以我覺得整部作品還是很積極的。

而且我覺得這段自白，多少也說出了自己的感受吧。我常常問起朋友“是在哪一刻覺得自己其實很普通”。我一直到大三快結束時才嚴肅地覺得自己只是個普通人，也許做不了什麼大事，但潛意識里仍然覺得還是要有一番作為，無論大小，至少無愧於己。

最近溫習魔法溫習到腦子都有點銹了，東想西想，有時候氣餒到覺得自己無論做什麼都無法達到自己的標準，很是洩氣，不知道之前做過的事都算些什麼，只是被生活推著走嗎。其實看電影看劇也並不是那麼開心和有趣，我覺得學習（廣義的）倒還更充實一些。

希望明天好起來。晚安。

大部分時候是自控且清醒的，除了午睡起來后那一個小時。有時候感到，和想成為的樣子，實在離得太遠，就會立刻感到沮喪，但又並不會放棄努力。

  • 早上完成了會計的所得稅會計的導圖重建，然後開始摸索hexo的一些組件（摸魚），有時候就是因為學得太開心就會得意忘形是怎麼個設定……
  • 午覺睡過頭
  • 然後下午一直開始surfing到晚上（居然重溫了童年時期看過的電視劇截圖，人無聊起來還真是蠻無聊的）
  • 晚飯前做了瑜伽
  • 晚上看著講義覺得沒什麼想複習的，倦怠max

好可怕，感到兩個月之後我會後悔而死。這段時間識別出最多的情緒就是，煩躁，倦怠和自責。一直在思考是什麼讓我在四年大學生活中保持著良好的自控能力，又是什麼讓我在大四這年失去這自控力

左思右想，是缺乏一個「讓我感覺」很酷的奮鬥目標吧。

大三中後期，在和GMAT死磕完之後，開始逐漸改變原來死板的作息，變得更加靈活，偶爾也會做計劃之外的事，變得不再每天戰戰兢兢追著進度跑。收穫嘛，大概是撿回了繪畫，看了很多好電影。和朋友的交流互動也越來越多了，甚至都變得有些不習慣一個人獨來獨往，也時常希望有人陪著。從前無法可想和人如此親近。

之後就是接連不斷的實習生活，從一份切換到另一份，適應了鬆散的不自省的生活。感到再難回復從前的狀態了。怎麼說，感到是自掘墳墓。想起蝙蝠俠黑暗騎士三部曲裡面，主角在最後說，小丑還是勝利了，因為他讓一個正義的人從自己內部瓦解了正義。

不能東想西想，至少目前要守護僅有的行動力，堅持早上四個小時的學習時間。

晚安。
