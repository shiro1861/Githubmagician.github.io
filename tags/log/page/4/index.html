因为看得太抑郁，打算看点亲和的书，于是转战《给青年诗人的信》。不知道看的什么译版，觉得用词和语句冗杂且难以理解，可能因为我不是青年诗人，不知道原文究竟是怎样的，不过好在篇幅比较短不至于难以忍受。整本书充满了那种虽然令人感到莫名其妙但读起来又觉得似曾相识说出了心中所想的句子，比如：

这是可笑的。我在这地坐在我的小屋里，我，布里格，已经是二十八岁了，没有人知道我这个人。我坐在这里，我是虚无。然而这个虚无开始想了，在五层楼上，一个灰色的巴黎的下午，它得出这样的思想：这是可能的吗，它想，人们还不曾看见过、认识过、说出过真实的与重要的事物？这是可能的吗，人们已经有了几千年的时间去观看、沉思、记载，而他们让这几千年过去了像是学校里休息的时间，在这时间内吃了一块黄油面包和一个苹果？ 是的，这是可能的。

好的，用短暂的复习前20分钟写了一些读后感，复习去也。

用了一个失眠的晚上看完了《给青年诗人的信》之后，因为想看些催眠的书，转战梁启超的《李鸿章传》，两日后进度50%，目前的观感是梁启超写得很够味，好肤浅的观感……好吧，其实观感是，“世事波上舟，沿洄安得住。”也是看得好抑郁，下本书真想看点欢快的。

魔法研习进度

这周五做了短暂休整，再次研究了有关记忆方法的文章，打算对学习方法进行调整。材料综合看了知乎上面一些大牛的经验，然后下载了一份有关记忆宫殿的学习材料，花了大约一个下午整理和阅读，并选取了部分来试验。整个过程还有点热血沸腾，莫名其妙……

目前最喜欢的还是串联记忆和联想记忆法，拿背诵内容多的审计实验了一下，比单纯背有意思多了，精神也更集中，因为要想莫名其妙的联想来记住。定桩和数字联想法有点不明所以，可能是对于我目前的备考科目不合适。

目前实践的心得：

  1. 记忆到考试就是“输入-编码-提取-运用”的过程，记不住实际上是编码和提取环节出现脱节
  2. 关于编码，就像是“制造路径”的感觉，我对应到使用思维导图整理这一步
  3. 提取就是以各种不同于初始输入的方式来复习
  4. 运用对于我就是做题

首先我改进记忆方法的目标（动机）是：

  • 第一目标：帮助我更高效地复习并通过以后所有我想通过的考试，一是专业知识，二是语言考试
  • 第二目标：在阅读和看电影时记忆更多的细节，一可写作时引用二可交谈时不至于空无一物
  • 第三目标：在将要开始的工作中能更快速地记下客户和同事的名字等细节以及对方交代的事务
  • 最高目标（误）：成为高段位学习者（坦白讲这个目标比拿下注册XXX考试的更令我热血沸腾是怎么回事……）

根据第一目标识别出的记忆客体特点：

  • 会计

    • 单纯记忆，概念性知识，分类标准等 （比如哪些交易计入什么科目）
    • 会计分录名称，需要一定理解
    • 一整套的处理思路，需要深入理解准则（某种交易从头到尾的处理）
    • 计算时易错的细节
    • 选择题快速计算的公式和技巧

  • 财务管理

    • 公式！！！
    • 解题思路
    • 单纯记忆，概念性知识，分类标准等
    • 计算时易错的细节

  • 税法

    • 征税对象税率征税时间计算公式
    • 计算细节

  • 审计

    • 单纯记忆，概念性知识等（应付简答题）
    • 需要深入理解并运用于判断的规则（比如选择什么审计程序，采用该程序或处理方法是否恰当）
    • 流程性质

  • 经济法就是往死里背吧（仲未复习）

  • 战略管理就是往死里背吧（仲未复习）

先写到这里，去干正事儿。突然感到谜之平静是怎么回事……

然后这几天复习进度都还算勉强踩着，审计一刷和财管二刷的进度差不多1.5天一章，包括复习讲义和自我归纳，会计卡死在负债和所有者权益那章，太庞杂了，正在一点点重新整理导图，税法卡死在企业所得税，也是庞杂……

反正现在的心情是很平静，感觉就像是准备一个庞大的期末考。上午基本可以保证复习效率，不过下午就有点悬。然后个人觉得自己的缺点就是太想一步到位，对自己每个复习环节要求很高，往往在开始前会觉得自己达不到而放弃去开始，造成拖延。下周的目标就是如何控制好下午的复习时间以及不要过于急进，学会分解任务，每次先做到及格。

magician今天有點長

7月 28, 2017

今早覺得頭痛，怎么都不想起床。堅持學了一個上午所有者權益和負債，實在不行了，於是休學罷工。發現現在真是沒有高中那股“讓我學我還能學啊啊啊啊還不能閉眼”的勁頭了。

下午看完了步履不停（歩いても 歩いても），看的是枝裕和導演的第一部片。原聲帶的吉他好聽，是夏天躺在陰涼的臥室里吹風扇的感覺。最後男主角的旁白令人有些難過，發現一個人的生老病死，說出來也就輕描淡寫一兩句話。

不過全片節奏對於我來說還是太緩慢。我也看過李安前期的“父親三部曲”（《推手》《飲食男女》《喜宴》），感覺要活潑很多:D，戲劇衝突感更強，令我在家庭劇中找到了意外的興趣點。

之後本來想重看銀翼殺手（Blade Runner），覺得太費神就刷完了涼宮春日的憂鬱（涼宮ハルヒの憂鬱）。这是我初中的時候差點就看了的作品。雖然自詡從中二少年圈退休多年，明知人生不复少年时，还是会看得热血沸腾，也还是希望自己能够回去。

其實也很久沒看動畫了，只是最近温习魔法太苦悶，挖了些初高中時期仰慕已久的舊作出來看。我看涼宮覺得比較有共鳴是在第十集左右。女主角涼宮有段自白大意是，曾經以為自己多麼獨特，但萬千世界人生海海，發現自己也只是普通人而已。發現這點之後，覺得人生陷入了憂鬱，所以才會希望每天的生活發生一些有趣的事。

這段自白，真是中二病開山鼻祖般的發言，我覺得說出了我理解的日本動畫的核心觀點之一──就算我獨特的存在與世界相矛盾，我也絕不妥協。雖然我個人對涼宮霸道的人設有一點不接受，但这个角色的魅力，就像片中男主角阿虛所說的“你們只會對現狀抱怨不滿，但春日是真的行動起來去改變這一切了啊。”
所以我覺得整部作品還是很積極的。

而且我覺得這段自白，多少也說出了自己的感受吧。我常常問起朋友“是在哪一刻覺得自己其實很普通”。我一直到大三快結束時才嚴肅地覺得自己只是個普通人，也許做不了什麼大事，但潛意識里仍然覺得還是要有一番作為，無論大小，至少無愧於己。

最近溫習魔法溫習到腦子都有點銹了，東想西想，有時候氣餒到覺得自己無論做什麼都無法達到自己的標準，很是洩氣，不知道之前做過的事都算些什麼，只是被生活推著走嗎。其實看電影看劇也並不是那麼開心和有趣，我覺得學習（廣義的）倒還更充實一些。

希望明天好起來。晚安。
